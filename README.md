# skills

Custom slash commands for [Claude Code](https://claude.ai/code).

Inspired by [Matt Pocock's skills repo](https://github.com/mattpocock/skills).

Each skill lives in its own folder with a `SKILL.md` file. Copy any skill folder into `~/.claude/skills/` to use it.

| Skill | Description |
|-------|-------------|
| [seo](./seo) | Full SEO, accessibility, and social preview audit |
| [hackathon-radar](./hackathon-radar) | Find open crypto/AI hackathons and get build ideas for each one |
| [log](./log) | Rolling session log -- saves detailed recaps so Claude picks up where you left off |
| [tldr](./tldr) | Quick session summary printed in conversation -- no files written |
